# CodeAlpha_task
Static Structural Analysis of an I-Beam | SolidWorks Simulation
Performed a complete Finite Element Analysis (FEA) of an I-type beam under a 1,000 kgf point load using SolidWorks Simulation 2021.
📌 Simulation Setup:
Material: AISI 304 Stainless Steel
Fixture: Fixed support at both ends
Load: 1,000 kgf (per item) applied on top flange
Mesh: High-quality solid mesh
📊 Key Results:
🔴 Max von Mises Stress: 3.053 × 10⁷ N/m² (30.53 MPa)
🟢 Yield Strength: 2.068 × 10⁸ N/m² (206.8 MPa)
✅ Minimum Factor of Safety: 6.8 — well within safe limits
Deformation Scale: 1,244.62x (for visualization)
